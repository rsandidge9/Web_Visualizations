WEATHER!!!

Summary: Lattitude vs. 'x'

Everyone knows that as you travel closer to the equator the temperature gets hotter.

This project was used to prove it!

A python script was created to visualize the weather of 500 cities across the world with varying distances from the equator. Data from OpenWeatherMap API was also used to popluate factors such as: temperature, humididty, cloudiness and wind speed.

Once all of the data was assembled. MatplotLib was used to plot different aspects of the weather in relation to latitude. This site provides the visualizations from this dataset that were created as part of the analysis. It aslo provides explanations of any trends and/or correlations that were found.

Scatter plots are used to interpret trends by examining the relationships between the two variables X(latitude) and Y (Temperature, Humidity, Cloudiness and Wind Speed). Analyze the scatter plot and determine if there are any distinct patterns: positive relationship, negative relationship and no relationship. However, just because the graph shows correlation does not always mean there is a relationship. Correlation does not equal causation.

To determine if there is a correlation between the two variables the r-value must be looked at and evaluated. The r-value is always between 1 and -1. If the r-value is exactly 1 then there is a perfect correlation between the two variables. As the number approaches 0 so does the likelihood of cause and effect.

Regression analysis is the standardized way of finding correlation between data sets and it also gives a predictive model.
